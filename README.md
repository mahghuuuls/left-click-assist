# Left Click Assist

Left Click Assist lets you break blocks without holding the left mouse button the whole time.

Bind the mod key in the Controls menu, hold the tool or item you want to use, and press the key once to arm the mod for that item. A small HUD component appears in the bottom-right corner with the armed item icon and an `ON` or `OFF` label. While that item is selected, left-click once to turn automatic block breaking on, and left-click again to turn it off.

The mod is focused on block breaking, not general autoclicking. Aiming at entities, air, fluids, or unbreakable blocks does not trigger automated block breaking, and manual attacks still work normally when you intentionally left-click an entity.

If you switch away from the armed item, auto click turns off. The HUD stays visible while the mod is still armed, so you can switch back to the original item and left-click again when you want to continue. Pressing the mod key again disarms it completely and hides the HUD component.

## Notes

- It requires a non-empty main-hand item when you arm it.
- Automatic block breaking does not work in Creative mode.

## Configuration

- The toggle key (unbound by default).
- The HUD component position and scale.

## Multiplayer

This mod is client-side and does not need to be installed on the server.

Some servers may consider automatic block breaking to be cheating or otherwise against their rules. Check the rules of the server you play on.
